# 用户使用手册

欢迎来到 OpenList！这是一个简单易用的文件管理工具，帮助您轻松管理云端存储的文件。本手册将指导您完成基本操作，让您快速上手。

## 登录系统

1. 打开浏览器，访问管理员提供的 OpenList 地址
2. 输入您的用户名和密码
3. 点击"登录"按钮进入系统

**提示**：首次登录后，建议在个人设置中修改默认密码以确保账户安全。

## 上传文件

### 方法一：网页直接上传

1. 在文件列表中，导航到您想要上传文件的目标文件夹
2. 点击页面右上角的  **"上传"**  按钮
3. 选择"上传文件"或"上传文件夹"
4. 从您的电脑中选择要上传的文件

**快捷操作**：您也可以直接将文件从电脑拖拽到浏览器窗口中进行上传。

### 方法二：通过 WebDAV 挂载

1. 获取 WebDAV 地址：`http://您的OpenList地址/dav`
2. 在电脑上映射网络驱动器：
    - **Windows**：打开"此电脑" → "映射网络驱动器" → 输入 WebDAV 地址
    - **macOS**：Finder → 按下 `Cmd + K` → 输入 WebDAV 地址
3. 输入您的 OpenList 用户名和密码
4. 连接成功后，即可像操作本地文件一样管理云端文件

## 下载文件

- **单个文件**：直接点击文件名即可开始下载
- **多个文件/文件夹**：
    1. 勾选需要下载的文件或文件夹左侧的复选框
    2. 点击顶部工具栏的  **"下载"**  按钮
    3. 系统将自动打包为 ZIP 格式后下载

## 文件管理

OpenList 提供丰富的文件管理功能：

- **重命名**：右键点击文件 → 选择"重命名" → 输入新名称
- **移动/复制**：选中文件 → 点击"移动"或"复制" → 选择目标文件夹
- **删除**：选中文件 → 点击"删除" → 确认删除操作
- **新建文件夹**：点击"新建" → 输入文件夹名称 → 确认创建

## ️ 在线预览

OpenList 支持多种文件格式的在线预览，无需下载即可查看：

- **图片**：JPG, PNG, GIF, BMP 等
- **视频**：MP4, AVI, MOV 等（支持在线播放）
- **音频**：MP3, WAV, FLAC 等（支持在线播放）
- **文档**：PDF, TXT, Markdown 等
- **Office 文件**：Word, Excel, PowerPoint（在线渲染预览）

## 分享文件

轻松创建分享链接，与他人共享文件：

1. 选中要分享的文件或文件夹
2. 点击顶部工具栏的  **"分享"**  按钮
3. 设置分享选项：
    - 访问密码（可选）
    - 有效期（可选）
4. 点击"生成链接"
5. 复制链接和提取码（如有）发送给他人

## 搜索文件

快速定位您需要的文件：

1. 在页面顶部的搜索框中输入关键词
2. 按下回车键或点击搜索图标
3. 系统将在当前存储源或文件夹内搜索并显示结果

## 常见问题

### 上传速度慢怎么办？

- 检查您的网络连接
- 尝试在非高峰时段上传大文件
- 考虑使用 WebDAV 方式上传，通常更稳定

### 大文件上传失败？

- Web 端上传超大文件可能受浏览器限制
- 建议使用 WebDAV 方式挂载上传
- 检查云盘服务商对单个文件大小的限制

### 如何修改密码？

1. 登录系统后，点击右上角的用户头像
2. 选择"个人设置"
3. 在"安全设置"中修改密码

### 忘记密码怎么办？

请联系您的系统管理员重置密码。
